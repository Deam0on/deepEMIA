Quick Start

Get started with deepEMIA in a few simple steps:

  1. Upload Data: Go to Files tab to upload training or measurement data
  2. Run Tasks: Use Tasks tab to execute inference, training, or evaluation
  3. View Results: Check Results tab for outputs and visualizations
  4. Manage Datasets: Use Datasets tab to create or modify dataset configurations
